• Numbered lists: where each item in the list is numbered.
  • Bullet lists: are lists that begin with a bullet point
  • Definitions lists: are made up of a set of terms along with the definitions for each of those terms

Numbered lists

Html:

          <ol>
            <li></li>
            ...
            <li></li>
          </ol>

Result:

  1. item-1
    2. ...
  2. item-n

Bullet lists

Html:

          <ul>
            <li></li>
            ...
            <li></li>
          </ul>

Result:

  • item-1
    • ...
  • item-n

Definitions lists

Html:

          <dl>
            <dt></dt>
            <dd></dd>
            ...
            <dt></dt>
            <dd></dd>
          </dl>

Result:

The definition term A
contain the definition A
...
The definition term Z
contain the definition Z
